## 🚀 部署说明

### Vercel部署步骤
```bash
cd ai-gift-song
# 1. 安装Vercel CLI
npm i -g vercel

# 2. 登录
vercel login

# 3. 部署
vercel

# 4. 设置环境变量（在Vercel Dashboard）
# SUNO_API_KEY = your_api_key

# 5. 生产环境部署
vercel --prod
```

### 环境变量
| 变量名 | 说明 |
|--------|------|
| `SUNO_API_KEY` | Evolink Suno API密钥 |
